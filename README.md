## Malicious Windows DuckyScripts

Here is a list of the malicious Windows DuckyScripts included in this repository:

*   **background_website.txt**: Opens a website in the background.
*   **change_background.txt**: Changes the desktop background to a solid color.
*   **create_fake_admin.txt**: Creates a new administrator account with the username "backdoor" and password "p@ssword123". (Requires administrator privileges)
*   **delete_desktop.txt**: Deletes all files on the desktop.
*   **disable_firewall.txt**: Disables the Windows Firewall. (Requires administrator privileges)
*   **disable_windows_defender.txt**: Disables Windows Defender. (Requires administrator privileges)
*   **rotate_screen.txt**: Rotates the screen.
*   **force_update.txt**: Fakes a Windows update screen.
*   **fork_bomb.txt**: Opens a large number of command prompts, potentially crashing the system.
*   **get_wifi_passwords.txt**: Extracts and saves Wi-Fi passwords. (Requires administrator privileges)
*   **show_desktop.txt**: Minimizes all windows to show the desktop.
*   **menacing_message.txt**: Opens the command prompt and displays a menacing message.
*   **random_website.txt**: Opens a random website from a predefined list.
*   **rick_roll.txt**: Opens the music video for "Never Gonna Give You Up" by Rick Astley.
*   **swap_mouse_buttons.txt**: Swaps the primary and secondary mouse buttons.
